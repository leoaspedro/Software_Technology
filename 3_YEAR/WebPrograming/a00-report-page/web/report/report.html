1. Explain how the scope works in JavaScript, how hoisting affects it and how a closure works.

In Javascript the scope determines the accessibility of variables, this means if you define a variable inside a function you will not be able to access it from outside the function but for example, if you declare a global variable ( using {}) you can access it everywhere in your code, despite being vulnerable to bugs, that's why you should always declare local variables. In Javascript you should always declare your fucntions before you use them so it gets hosted properly. A closure is when you have a function within another function, so the closure will be the inner function.
